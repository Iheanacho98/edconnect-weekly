Project Abstract

This purpose of the e-learning application is to provide an environment for conducive learning on digital platforms. This study shows effective ways in learning can take place. Learning will be administered through Videos Lectures, Research, Solutions and Activism. Users should be able to learn anything through watching videos. They should also be able to make experiments with lectures and discuss it on forums, and then improve the website by sharing better solutions. Furthermore, users should be able to learn more from other users when the website is shared with them.
